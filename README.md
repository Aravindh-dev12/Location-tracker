# ☀️ Solar Maintenance Tracking Application

## Installation

1. Import `database.sql` into MySQL
2. Edit `config.php` with your database credentials
3. Open `index.html` in browser
4. Login with: admin@solar.com / admin123

## Features

- WiFi + IP location tracking (no GPS)
- Real-time 3D map with MapLibre
- Work report submission
- Image upload
- Admin dashboard
- Employee app

## Default Accounts

- Admin: admin@solar.com / admin123
- John: john@solar.com / john123
- Sarah: sarah@solar.com / sarah123

## Structure

- `/api/` - Backend API endpoints
- `/admin/` - Admin dashboard
- `/employee/` - Employee app
- `/assets/` - JavaScript and CSS
- `/uploads/` - User files
